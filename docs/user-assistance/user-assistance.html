用户帮助¶

用户帮助对应英文的 User Assistance 或 User Support。产品在设计的时候，应该尽可能利用用户的心智模型，从而减少用户的学习成本，凭借用户的直觉就能完成产品的使用，但实际上，除了极个别常用的工具，功能较少的情况，绝大部分产品用户在使用的时候，都是需要帮助的。

技术文档¶

技术文档是常见的帮助形式，用户在使用前，可以通过官方文档学习后使用，这种方式并不为用户喜欢，因为并不符合用户的习惯。
