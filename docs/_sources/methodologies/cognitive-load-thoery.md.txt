# 认知负荷理论

认知负荷理论 (Cognitive Load Theory)，主要解决如何降低Working Memory 的负荷，从而能让知识更好的进入到 Long-term Memory中。

该理论主要应用在复杂认知的教程设计或者技术上难度较大的材料。



## Human Cognitive Architecture

人的认知架构，有三部分组成：

- 感官记忆 （sensory memory)
- 工作记忆  (Working Memory)
- 长期记忆 ( Long-term Memory)



### 长期记忆

长期记忆是一个精密的知识存储结构了，有了这些世界知识，我们才能够理解、思考和解决问题。也称为 **Schema**。Schema本身是终身学习的，而且Schema本身也是嵌套的。进入到长期记忆，才能称为我们的知识。



### 工作记忆

工作记忆是有认知极限的。认知极限是指工作记忆一次能处理的信息块。目前的研究结论是，每次可以处理的信息块 (chunks) 是7 +/- 2。



感兴趣的读者，可以自己做一些实验。
